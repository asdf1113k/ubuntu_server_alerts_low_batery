# для кого это?
у вас есть ноутбук и на нём стоит ubuntu сервер, и вы хотите чтоб он оповещал вас о низком заряде - этот скрипт точно для вас

# как работает скрипт
если 20 процентов заряда и зарядное устройство не подключено
воспроизводиться звук оповещения о низком заряде "low battery"

# заметки:
переделать скрипт с ЯП python на ЯП bash, чтоб уменьшить зависимости!!!

# install and run
для работы скрипта требуеться:

**python>=*3.10*** *в дальнейшем переделаю на ЯП BASH*

---
переходим в каталог пользователя
```bash
cd ~
```
---
клонируем репозиторий
```bash
git clone https://github.com/asdf1113k/ubuntu_server_alerts_low_batery.git
```
---
переходим в каталог репозитория
```
cd ubuntu_server_alerts_low_batery
```
---
запускаем автоматическую установку, скрипт(AutoStartScript.py) который сам добавит AlertLowBattery.py
в автозапуск systemctl --user
```
python3 AutoStartScript.py
```

# отслеживание работы скрипта
```sh
journalctl -f
```
покажет заряд и статус зарядного устройства

пример:
```sh
Mar 31 11:49:43 LAPTOP-2CDN6G4C python3[3358]: 100% статус зарядного устройства: 1 
```
