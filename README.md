# для кого это?
у вас есть ноутбук и на нём стоит ubuntu сервер, и вы хотите чтоб он оповещал вас о низком заряде - этот скрипт точно для вас

# как работает скрипт
если 20 процентов заряда и зарядное устройство не подключено
воспроизводиться звук оповещения о низком заряде "low battery"

# install
важно чтоб вы вошли как пользователь у которого это установлено

или как минимум запустите пользователя и не выйдете с него

команда для этого
```
sudo -iu $USER
```
если не войти, то при обращении к systemctl --user произайдёт ошибка
```
cd ~
git clone https://github.com/asdf1113k/ubuntu_server_alerts_low_batery.git
cd ubuntu_server_alerts_low_batery
chmod +x AlertLowBattery.py
python3 AutoStartScript.py
```

# отслеживание работы скрипта
```
journalctl -f
```
покажет заряд и статус зарядного устройства
